CSS Properties Assignment

1. CSS FONT PROPERTIES

Property Meaning Syntax Example Use Case
font-family Specifies the font of text. font-family: "font name", fallback; font-family: Arial, sans-serif; Used to set the typeface of headings or body text.
font-size Specifies the size of the font. font-size: size; font-size: 16px; Used to make text larger or smaller.
font-style Specifies the style of the font (normal, italic, oblique). font-style: normal | italic | oblique; font-style: italic; Used to italicize text.
font-weight Sets the weight (boldness) of the font. font-weight: normal | bold | 100–900; font-weight: bold; Used to make text bold or light.
font-variant Specifies whether text is displayed in small-caps. font-variant: normal | small-caps; font-variant: small-caps; Used to style text with capitalized small letters.
font Shorthand for setting font-style, font-variant, font-weight, font-size, line-height, and font-family. font: style variant weight size/line-height family; font: italic small-caps bold 16px/1.5 Arial; Used to set all font properties in one declaration.

2. CSS COLOR PROPERTIES

Property Meaning Syntax Example Use Case
color Sets the color of the text. color: color-name | hex | rgb | rgba; color: red;
color: #333333;		 Used to style the color of text in headings or paragraphs.
background-color Sets the background color of an element. background-color: color-name | hex | rgb | rgba; background-color: yellow;
background-color: #f0f0f0;		 Used to highlight or visually separate sections.
opacity Sets the transparency level of an element. opacity: 0.0 to 1.0; opacity: 0.5; Used to create transparency effects.

3.CSS BACKGROUND PROPERTIES

Property Meaning Syntax Example Use Case
background-color Sets the background color of an element. background-color: yellow; body { background-color: yellow; } Used to highlight or theme a section.
background-image Sets an image as the background. background-image: url('image.jpg'); div { background-image: url('bg.png'); } Used for decorative or branding purposes.
background-repeat Defines if/how the background image should repeat. background-repeat: no-repeat; div { background-repeat: no-repeat; } Prevents images from tiling.
background-size Specifies the size of the background image. background-size: cover; div { background-size: cover; } Used to make images fit the element area.

4. CSS TEXT PROPERTIES

Property Meaning Syntax Example Use Case
text-align Specifies the horizontal alignment of text within an element. text-align: left | right | center | justify; text-align: center; Used to align text centrally in headings or paragraphs.
text-decoration Defines decorations added to text such as underline, overline, or line-through. text-decoration: none | underline | line-through; text-decoration: underline; Used to style links or emphasize text.
text-transform Controls the capitalization of text. text-transform: uppercase | lowercase | capitalize; text-transform: uppercase; Used to change text case, for example in titles or buttons.
letter-spacing Specifies the space between characters in text. letter-spacing: value; letter-spacing: 2px; Used to increase or decrease spacing between letters for design effect.
line-height Sets the amount of space between lines of text. line-height: normal | number | length; line-height: 1.5; Improves readability by controlling line spacing in paragraphs.
word-spacing Specifies the space between words in a text. word-spacing: value; word-spacing: 4px; Used to adjust spacing for better text appearance.
text-indent Indents the first line of a text block. text-indent: value; text-indent: 30px; Used for paragraph formatting to indent the first line.

Property Meaning Syntax Example Use Case
border Sets the width, style, and color of an element's border. border: width style color; border: 2px solid black; Used to define the border around elements for emphasis or separation.

19. CSS PSEUDO-CLASSES

Property Meaning Syntax Example Use Case
:hover Applies a style when the user hovers over an element. selector:hover {style} button:hover {background-color: blue;} Used to add interactivity to buttons or links.
:active Applies a style when an element is being clicked. selector:active {style} a:active {color: red;} Used to show visual feedback when clicking.
:focus Applies style when an element gains focus (e.g., input). selector:focus {style} input:focus {border: 2px solid green;} Used to highlight inputs during form filling.
:first-child Targets the first child element within a parent. selector:first-child {style} li:first-child {font-weight: bold;} Used for specific styling of the first list item.
:nth-child(n) Targets the nth child element within a parent. selector:nth-child(n) {style} tr:nth-child(2) {background: gray;} Used in styling tables or lists selectively.

20. CSS PSEUDO-ELEMENTS

Property Meaning Syntax Example Use Case
::before Inserts content before an element’s actual content. selector::before {style} p::before {content: "* ";} Used to add decorative or structural text before elements.
::after Inserts content after an element’s actual content. selector::after {style} div::after {content: "✓";} Used for visual indicators like checkmarks or notes.
::first-letter Styles the first letter of a block of text. selector::first-letter {style} p::first-letter {font-size: 200%;} Used for typographic effects like drop caps.
::first-line Styles the first line of a block of text. selector::first-line {style} p::first-line {color: blue;} Used to emphasize the beginning of a paragraph.
::selection Applies style to text selected by the user. ::selection {style} ::selection {background: yellow;} Used to improve readability of selected text.
